rayriffy-de
===========

Transform your mistyped language text easily with the power of JavaScript

Requirements
------------

-   Node LTS

Installation
------------

```
yarn
```

Usage
-----

Start development server by

```
yarn dev
```

Build static site by

```
yarn build
```
